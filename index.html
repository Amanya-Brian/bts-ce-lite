Quick start

Multi-vendor/technology

Support for the leading vendors i.e. Ericsson, Huawei, ZTE, Nokia, and Motorola. And support for 2G, 3G, 4G, and 5G

Network dump processing

Parsing and loading multi network configuration and performance management dump formats

Reporting

Supports the creation of tabular, graphical and composite/dashboard-like reports

GIS

The GIS module displays and searches network entities with ease

v0.3.0-alpha.5 is available for testing

This is a pre-release version.

Process various network dump formats

Boda-Lite processes configuraiton management(CM), performance management(PM), inventory, and fault management dumps.

Creation of unlimited reports

You have the ability to create an unlimited number of reports of various kinds. Boda-Lite also comes with a number of in-built useful reports such as key cell parameters

Advanced GIS Module

Provides intuitive interface for searching network entities (such as cells and relations) and displaying their parameters.
